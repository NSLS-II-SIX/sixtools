============
Installation
============

#. Create and activate a conda environment following `NSLS-II Conda Instructions <https://nsls-ii.github.io/conda.html?highlight=conda/>`_

#. Install the software via the the command line::

    $ git clone sixtools
    $ cd sixtools
    $ python3 -m pip install -e . 

#. Looking at or using the example notebooks in the sixtools repository should be a good place to start.
